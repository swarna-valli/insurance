## 📂 Featured Projects

### [1. Actuarial Support: Rate Change Simulation](./Project-1-Rate-Simulation)
**Problem:** Heartland needed to predict the "Dislocation" (customer impact) of a 5% rate filing.
**Solution:** Built a SAS model to simulate premium changes across 50k active policies.
**Impact:** Identified 2% of policies with >15% increases, allowing for a "Loyalty Buffer" strategy.

### [2. Risk Management: Geographic CAT Exposure Mapping](./Project-2-Geo-Mapping)
**Problem:** High concentration of "Total Insured Value" (TIV) in flood-prone Ontario FSAs.
**Solution:** Engineered an SQL pipeline to bucket exposure by postal code (FSA).
**Impact:** Provided the Underwriting team with a heatmap to manage regional capacity.
